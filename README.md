# Task2_EDA_Business_Intelligence
Exploratory Data Analysis and Business Intelligence project on a sales dataset using Python, Pandas, Matplotlib, Seaborn, SQL, and VS Code. This project includes data analysis, visualizations, business insights, SQL queries, and reporting.

## Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on a sales dataset to identify trends, customer behavior, product performance, and business insights.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* SQL
* VS Code

## Dataset Features

* Customer Information
* Product Details
* Sales Data
* Quantity Sold
* Unit Price
* Total Sales

## Visualizations

* Category Distribution
* Gender Distribution
* Age Distribution
* Top Cities by Sales
* Top Products by Sales
* Monthly Sales Trend
* Correlation Heatmap

## Key Findings

* Electronics is the top-performing category.
* Laptop generated the highest revenue.
* Patna recorded the highest sales.
* Customer distribution is nearly balanced between males and females.
* Quantity and Unit Price strongly influence Total Sales.
